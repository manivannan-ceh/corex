# APKVault - APK Version Control Frontend

This is a modern, production-ready React frontend (UI only) for an APK Version Control system built with Vite, React, TypeScript, and Tailwind CSS.

## Features

- **Dashboard**: Overview of total projects, releases, and recent uploads.
- **Projects**: Manage and view all your APK projects.
- **Project Details**: View version history, share, and download releases.
- **Upload APK**: Drag-and-drop support with progress tracking to create new releases.
- **Share**: Generate temporary access codes and shareable links for releases.
- **Public Download**: A public-facing page to verify access codes and download APKs.
- **Authentication**: JWT-based authentication.

## Tech Stack

- **React** (Vite)
- **TypeScript**
- **Tailwind CSS**
- **Axios**
- **React Router**
- **React Hot Toast**
- **Lucide React** (Icons)

## Setup and Running

1. **Install Node.js**: Ensure you have Node.js and npm installed.
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Start Development Server**:
   ```bash
   npm run dev
   ```

## Design Notes

The UI uses a dark-mode first design, featuring glassmorphism elements, subtle animations, and a sleek layout inspired by modern platforms like Vercel and Linear.
